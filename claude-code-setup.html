Get Claude Code running on Windows in 5 minutes

1

Install Node.js

Claude Code requires Node.js 18 or higher. Download and install it first:

üì• Download Node.js (LTS)
üí° Tip

Choose the LTS (Long Term Support) version. During installation, check the box to install necessary tools.

After installation, verify it worked by opening PowerShell and running:

PowerShell
node --version

You should see something like v22.x.x

2

Install Claude Code

Open PowerShell as Administrator (right-click ‚Üí Run as Administrator) and run:

PowerShell (Admin)
npm install -g @anthropic-ai/claude-code

If you get permission errors, try running PowerShell as Administrator, or use this command first:

3

Authenticate with Anthropic

You need an Anthropic API key or a Claude Max subscription. Choose one:

Option A: API Key (Pay-as-you-go)

Get your API key from the Anthropic Console:

üîë Get API Key

Then set it in PowerShell:

PowerShell
$env:ANTHROPIC_API_KEY = "sk-ant-your-key-here"

To make it permanent, add to your PowerShell profile:

PowerShell - Add to profile
notepad $PROFILE

Add this line to the file that opens:

Add to profile file
$env:ANTHROPIC_API_KEY = "sk-ant-your-key-here"

Option B: Claude Max Subscription (Browser login)

If you have Claude Max ($100/mo or $200/mo), just run Claude Code and it will open a browser to authenticate:

PowerShell
claude
4

Run Claude Code

Navigate to a project folder and start Claude Code:

PowerShell
cd C:\Users\YourName\Projects\my-project claude
üéâ You're in!

Claude Code will start in interactive mode. Type your requests and Claude will help you code!

Troubleshooting

‚ùå "claude is not recognized"

Node.js bin folder isn't in your PATH. Restart PowerShell or run:

PowerShell
$env:Path = [System.Environment]::GetEnvironmentVariable("Path","Machine") + ";" + [System.Environment]::GetEnvironmentVariable("Path","User")

‚ùå Permission denied errors

Run PowerShell as Administrator and try:

PowerShell (Admin)
npm cache clean --force npm install -g @anthropic-ai/claude-code

‚ùå API key not working

Make sure the key starts with sk-ant- and has no extra spaces.
