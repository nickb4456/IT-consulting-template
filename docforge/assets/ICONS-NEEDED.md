# Icon Assets Required

The manifest.xml references these icon files that need to be created.

## Required Icons

|------|------|-------------|
| `icon-16.png` | 16x16 | Toolbar small |
| `icon-32.png` | 32x32 | Toolbar standard |
| `icon-64.png` | 64x64 | High-res toolbar |
| `icon-80.png` | 80x80 | Gallery icon |

## Design Guidelines

- **Style:** Fluent UI / Office design language
- **Colors:** Use neutral grays, accent blue (#0078D4)
- **Background:** Transparent
- **Format:** PNG with alpha channel

## Quick Placeholder Option

- Blue (#0078D4) for main icons
- Green (#107C10) for fill icons
- Orange (#CA5010) for numbering icons
- Purple (#8764B8) for header icons
